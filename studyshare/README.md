# StudyShare - Frontend

Proyecto: red social académica para compartir tareas y pedir ayuda.

## Estructura
- src/
  - components/
  - pages/
  - redux/
  - routes/
  - firebaseConfig.js
  - styles/

## Setup
1. Clona el repo.
2. `npm install`
3. Configura `src/firebaseConfig.js` con tus credenciales Firebase.
4. `npm start` para desarrollo.

## Deploy (Netlify)
1. Conecta tu repo en Netlify.
2. Build command: `npm run build`
3. Publish folder: `build`
